# 笔记
## Node运行时
* Node API: JavaScript实用程序(如文件和网络I/O)以及其他许多实用程序(如加密和压缩)
* Node Core: 一组JavaScript模块，用于实现Node API。
  * JavaScript引擎: Chrome的V8引擎: 一个快速的JavaScript代码编译程序，用于加载、优化和运行JavaScript代码
  * 事件循环：使用称为libuv的事件驱动型非阻塞I/O库来实施，使其实现轻量化和高效
## Node API
* 文件系统
* 加密
* 事件
* HTTP
* 流
